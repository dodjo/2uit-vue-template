Как работать с шаблоном?

1. Склонировать
2. Удалить папку .git
3. Сделать npm i 
4. npm build
5. npm serve
6. Изменить стили по своему дизайну.
7. Определиться, что из имеющегося в шаблоне необходимо в проекте.
8. Удалить ненужные файлы.
